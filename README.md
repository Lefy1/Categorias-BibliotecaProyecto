# Categorias-BibliotecaProyecto
Microservicio que contiene catálogo de categorias del proyecto de biblioteca para asignatura Desarrollo Fullstack I.

URL para obtener todos las categorias en postman: GET localhost:8083/api/v1/categorias

URL para obtener una categoria por ID en postman: GET localhost:8083/api/v1/categorias/{n° de id}

URL para agregar una categoria en postman: POST localhost:8083/api/v1/categorias

URL para actualizar una categoria en postman: PUT localhost:8083/api/v1/categorias/{n° de id}

URL para eliminar una categoria en postman: DELETE localhost:8083/api/v1/categorias/{n° de id}
